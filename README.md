## üíª Installation & Usage

```bash
# Clone the repository
git clone https://github.com/jonkllfaton/React-landing-page-suite.git

# Navigate to the project folder
cd React-landing-page-suite

# Install dependencies
npm install
# or
yarn install

# Start the development server
npm start
# or
yarn start
```

Open your browser at `http://localhost:3000` to view your landing page in action.

## üöÄ Customization Guide

1. **Components:** Modify `src/components` to update sections
2. **Styles:** Edit `src/styles` for colors, fonts, and layout adjustments
3. **Content:** Change JSX files for text, images, and links
4. **Build & Deploy:** Run `npm run build` or `yarn build` for production-ready files
