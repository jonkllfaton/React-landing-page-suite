## ğŸ’» Installation & Usage

```bash
# Clone the repository
git clone https://github.com/jonkllfaton/React-landing-page-suite.git

# Navigate to project folder
cd React-landing-page-suite

# Install dependencies
npm install
# or
yarn install

# Start the development server
npm start
# or
yarn start
```

Open `http://localhost:3000` to see your landing page in action.
